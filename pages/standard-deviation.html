Standard Deviation

Definition

Standard Deviation is a numerical representation of how data values are spread in reference to the mean of the data values. It is the square root of the variance of a data set.

R Code

To find the standard deviation of a data set using R we use the function sd().

> sd(classheights)

In this example, classheights is the name of a set of values representing all of the heights of students in Stat 151.This should be replaced with the data set for which you would like to find the standard deviation.
