Mean

Definition

The mean is a numerical measure of the central location of a group of data values. It is the sum of all values divided by the data count.

R Code

To find the mean of a data set using R we use the function mean().

> mean(classheights)

In this example, classheights is the name of a set of values representing all of the heights of students in Stat 151.This should be replaced by whatever data set for which you would like to find the mean.
